# Real Time Auction Platform

A fullstack realtime auction platform where users can register, login, create auction, and place bids in real time.

The application uses Sockets to enable live communication between users, allowing bids and auction updates to be reflected instantly across all connected clients.

---

# Features

- User registration and authentication (cookie-based session handling)
- Create and manage auction listings
- Place bids on active auctions (excluding own listings)
- Real-time bid updates using Sockets
- Countdown and auction expiration logic
- View active auction details in real time

# Tech Stack

- Node.js
- Express
- TypeScript
- Socket.io
- Cookies for authentication
- Frontend: JavaScript / TypeScript / HTML / CSS
